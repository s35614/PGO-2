
# PGO-2 MiniBiblioteka

Projekt przedstawia prosty system biblioteki napisany w Javie.

## Klasy

- Ksiazka

- Czytelnik

- Biblioteka

- Main

## Funkcje

- dodawanie ksiazek

- wypisywanie dostepnych ksiazek

- wyszukiwanie ksiazki po tytule

- wypozyczanie ksiazek

- zwracanie ksiazek

## Scenariusz testowy

W pliku Main utworzono przykladowe ksiazki i czytelnikow oraz pokazano:

- dodanie ksiazek do biblioteki

- wypozyczenie ksiazki

- zwrot ksiazki

- liczenie dostepnych ksiazek
